PRIPREMA ZA VJEŽBU:

1. Što znate o RGBA bojama? RGBA (Red, Green, Blue, Alpha) su boje za web dizajn koje su definirane s četiri komponente: crvene, zelene, plave i alfa kanal (prozirnost). Svaka komponenta je vrijednost između 0 i 255. 2.Na koje se načine mogu zadavati boje pri pisanju CSS pravila? Postoje nekoliko načina za zadavanje boja u CSS-u:

Hexadecimalni kodovi: ovaj format boja koristi šest znamenaka koje predstavljaju količinu crvene, zelene i plave boje (npr. #FF0000 za crvenu boju).
RGB (Red, Green, Blue) vrijednosti: ovaj format boja koristi tri vrijednosti od 0 do 255 koje predstavljaju količinu crvene, zelene i plave boje (npr. rgb(255, 0, 0) za crvenu boju).
RGBA (Red, Green, Blue, Alpha) vrijednosti: ova varijanta RGB formata koristi četiri vrijednosti (crvene, zelene, plave i alfa kanal za prozirnost) (npr. rgba(255, 0, 0, 0.5) za poluprozirnu crvenu boju).
Imenovane boje: neke standardne boje (kao što su crvena, plava, zelena itd.) mogu se zadati i pomoću njihovog imena (npr. "red" ili "blue").
HSL (Hue, Saturation, Lightness) vrijednosti: ovaj format boja koristi tri vrijednosti za definiranje nijanse, zasićenosti i svjetline boje (npr. hsl(0, 100%, 50%) za crvenu boju).
