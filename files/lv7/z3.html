DESET NAJPOPULARNIJIH PROGRAMSKIH JEZIKA

1. JavaScript

Svaki moderni website koristi JavaScript, nije na odmet znati taj programski jezik.
UPOTREBA: Razvoj interaktivnih HTML.stranica

2.Python

Ima jednostavnu sintakstu, iznimno je čitljiv i samim time početnicima idealan.
UPOTREBA: Web aplikacije, korisnička sučelja, analizu podataka, statistike...

3. Java

Jedan od najpopularnijih programskih jezika i onaj koji svi moraju znati ukoliko se žele baviti ovim poslom.
UPOTREBA:Koristi se za sve, od weba do Android aplikacija za pametne telefone i tablete.

4. Ruby

Programski jezik koji omogućava brzu izradu web aplikacija.
UPOTREBA:Sjajan za kickstart projekte i početnicima savršen, ukoliko je cilj raditi s aplikacijama ili raditi projekte.

5. PHP

Programski jezik koji služi za rad s MySQL i za stvaranje aplikacija koje rade s podacima.
UPOTREBA:S WordPress je primjer velikog sustava nastalog na PHP osnovama.

6. C++

Iako je to za "niže razine", potrebno je znanje ovog programskog jezika ako se želite spojiti direktno na hardver i izvući što je moguće više snage.
UPOTREBA: Odličan programski jezik za stvaranje moćnog desktop softvera, hardvera koji izvlači maksimum pa je gejmersko iskustvo bolje, a služi i za aplikacije za pametne telefone, konzole i desktop računala. Naravno, one vezane uz memoriju i njeno bolje iskorištavanje.

7. CSS

U široj namjeni služi za dizajniranje web stranica i aplikacija koje se baziraju na internetskim preglednicima.
UPOTREBA:Koristina za one koji rade s (ili za) više platformi.

8. C#

Osnove, osnove! Svi za njega znaju, svaki programer upućen je barem donekle u njega.
UPOTREBA:Da biste mogli raditi s Microsoft platformama morate biti detaljno upućeni u ovaj programski jezik.

9. Go

Ovdje je riječ o Googleovom programskom jeziku otvorenog tipa, idealnog za razvoj web aplikacija s minimalnim frameworkom, web servere, aplikacijska sučelja...
UPOTREBA:Developeri tvrde da tko savlada Javascript i(li), HTML te PHP, Go će naučiti kao od šale. Živi bili pa vidjeli!

10. C

Ovaj programski jezik je popularan još uvijek jer je malen, brz i moćan.
UPOTREBA:Ukoliko želite najviše za resurse kojima raspolažete, C je odgovor.
